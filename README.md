# 🏍️ Bike Life

A simple ride-logging web app for motorcyclists.

## Features

- Log rides with destination, distance, date, and notes
- Save rides locally (in browser)
- View ride history

## Tech Stack

- HTML/CSS/JavaScript
- LocalStorage for data
- Hosted on Vercel
